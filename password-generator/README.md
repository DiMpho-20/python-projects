# 🔐 Password Generator

A command-line password generator built with Python that creates random passwords based on user preferences.

Users can choose the password length and whether to include uppercase letters, numbers, and symbols.

---

## ✨ Features

- 🔢 Choose password length
- 🔠 Include uppercase letters
- 🔡 Include lowercase letters
- 🔢 Include numbers
- 🔣 Include special symbols
- 🎲 Generate a random password

## ⚙️ How It Works

1. The user enters the desired password length.
2. The user chooses whether to include:
   - Uppercase letters
   - Numbers
   - Symbols
3. The program builds a character pool based on the selected options.
4. A random password is generated and displayed.
